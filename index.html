Image-Text Pairing Validation Tool for TTB Labels (Demo)
Input Files



Upload one or more image-text pairs with the same base name (e.g., label_1.png and label_1.txt). Jobs without a matching set of image and text names will not process.

Acceptable image formats are .PNG, .JPEG, and WebP. Acceptable text formats are .TXT and .CSV.

All text files should be formatted with exactly seven comma-separated values matching the application information form in this order: 1. Brand Name, 2. Class/Type, 3. Alcohol Content, 4. Net Contents, 5. Bottler/Producer Address, 6. Import Country, 7. Government Warning present? (yes or no). Any fields not required should be marked na. Fields containing commas should be wrapped in " ".

Example of text file contents:
Malt & Hop Brewery,Pale Ale,5% ALC/VOL,1 PINT 8 FL OZ,"Malt & Hop Brewery, Hyattsville, MD",na,yes
